# Previews
Is a small script to display the website designs in the browser.    
A lot of our customers were confused how their jpegs will look as website.

## Build
	// installs dev-dependencies 
    npm install
	// compiles less and typescripts
	gulp

## Edit
	// installs dev-dependencies 
	npm install
	// installs typings
	typings install
	// starts localhost
	gulp serve
